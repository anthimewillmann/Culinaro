# Culinaro

Culinaro is an iOS recipe and cooking app built with Swift.

The app focuses on a clean and intuitive cooking experience, allowing users to discover, organize, and manage recipes in a modern mobile interface.

## Features

- Browse and manage recipes
- Save favorite meals
- Ingredient and meal organization
- Modern SwiftUI-based interface
- Native iOS experience
- Fast and lightweight design

## Tech Stack

- Swift
- SwiftUI
- Xcode
- iOS Development
