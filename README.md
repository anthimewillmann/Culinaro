# Culinaro

Culinaro is an iOS recipe and cooking app built with Swift.

The app focuses on a clean and intuitive cooking experience, allowing users to discover, organize, and manage recipes in a modern mobile interface.

  </a>
</p>

## Features

- Browse and manage recipes
- Save favorite meals
- Ingredient and meal organization
- Modern SwiftUI-based interface
- Native iOS experience
- Fast and lightweight design

## Tech Stack

- Swift
- SwiftUI
- Xcode
- iOS Development

## Installation

```bash

git clone https://github.com/anthimewillmann/Culinaro.git

```

1. Open the project in Xcode
2. Build and run on an iOS simulator or device

## Motivation

Culinaro was created as a personal iOS development project to improve my skills in:
- Swift development
- UI/UX design
- App architecture
- Mobile application workflows

The goal was to build a clean and practical cooking app with a focus on usability and modern design.
